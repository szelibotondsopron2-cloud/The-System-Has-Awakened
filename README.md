# Calis10x System

Native Android self-improvement game app.

Includes:
- dark navy System-style UI
- workout tracking
- daily quests
- trophies/ranks
- punishment wheel
- store
- local saving
- overlay restriction service
- Usage Access / Display Over Apps permission flow
- GitHub Actions APK builder

## Build APK with GitHub

Upload the unzipped contents of this folder to your GitHub repository.

Then go to:

**Actions → Build Android APK → Run workflow**

After it finishes, download the artifact:

**Calis10x-debug-apk**

Unzip it and install the APK on your Android phone.
